Child Theme Example for WordPress
----

Developed and owned by Željan Topić, http://bizzthemes.com/

Install
----
	1. Make sure that Bizznis parent theme is installed.
	2. Upload the theme folder via FTP to your 'wp-content/themes/' directory.
	3. Go to your WordPress dashboard and select Appearance.
	5. Inside your WordPress dashboard, go to Appearance > Theme Settings and configure theme to your liking.
